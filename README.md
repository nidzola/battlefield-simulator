## Battlefield simulator

Battlefield simulator v1.0,
tested with: Node.js v6.10.0

## Usage

```sh
# clone it:
git clone https://github.com/nidzola/battlefield-simulator.git
cd battlefield-simulator

# run it:
npm battle.js

# optional: 
Update config.js
```
